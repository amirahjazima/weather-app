## Features

- Clean responsive UI with CSS

## Tech Stack
- Node.js, Express
- EJS templating
- OpenCage API, Open-Meteo API
- dotenv for secrets

## Setup
1. Clone the repo
2. Run `npm install`
3. Add `.env` file with your API keys: OPENCAGE_API_KEY=your_api_key
4. Start the server: `npm start`
